# KURAKANI 💬

**KURAKANI** is a modern real-time one-to-one chat application built with the MERN ecosystem. It provides a fast, responsive messaging experience with secure authentication and instant communication powered by Socket.IO.

## Features

* 🔐 Secure JWT authentication
* 💬 Real-time one-to-one messaging
* ⚡ Instant message delivery with Socket.IO
* 🟢 Online user status
* 🔍 Search users and start conversations
* 🍪 Secure HTTP-only cookie authentication

## Tech Stack

### Frontend

* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS
* Framer Motion
* Socket.IO Client

### Backend

* Express.js
* Node.js
* TypeScript
* MongoDB
* Mongoose
* Socket.IO
* JWT Authentication
